## TooCool GasBot: Privacy policy

Welcome to the TooCool GasBot app for Android!

This is an Android app developed by TooCool Development Team which is available on Google Play.

We hereby state, to the best of our knowledge and belief, that we have not programmed this app to collect any personally identifiable information. All data (app preferences if any) created by the you (the user) is stored on your device only, and can be simply erased by clearing the app's data or uninstalling it. This application wirelessly acquires sensory data securely collected from our dedicated hardware installed on your site and stored on our secure database system. The application reads data from database server and manipulates data for meaningful display. 

For user authentication, only necessary information including email, uid, display name and photo's URL (if available) is required and needed to be recorded in our database system. The purpose of using this information is for secure login via Google authentication system in which all login processes is taken place on Google's authentication system side. 

### There is no special PERMISSION required to use this app.
